================================================================================
CLASHCODE - PROJECT OVERVIEW
================================================================================

WHAT IS CLASHCODE?
------------------
CLASHCODE is an industrial-grade, gamified coding platform designed for:
- Competitive programming
- Mastery-based progression
- AI-assisted tutoring
- Community engagement

Tagline: "Learn to Code Through Gamification"

================================================================================
SYSTEM ARCHITECTURE
================================================================================

Frontend: React 19 + Vite + Tailwind CSS + Zustand
Backend: Microservices architecture (4 services)
Gateway: Nginx reverse proxy
Database: PostgreSQL + Redis + ChromaDB + DynamoDB Local

Service Breakdown:
1. Core Service (Django 5) - Port 8000
   - User authentication & profiles
   - Challenge management
   - Store & payments (Razorpay)
   - Social features (posts, follows)
   - Admin dashboard
   - XP economy & achievements

2. Chat Service (FastAPI) - Port 8001
   - Real-time messaging via WebSockets
   - User presence tracking
   - Redis Pub/Sub for broadcasting
   - DynamoDB message persistence

3. AI Service (FastAPI) - Port 8002
   - RAG-powered AI tutoring
   - Contextual hints (3 levels)
   - Code analysis & suggestions
   - ChromaDB vector retrieval
   - LangChain integration

4. Executor Service (FastAPI) - Port 8011
   - Sandboxed Python code execution
   - Docker container isolation
   - AST validation
   - Resource limits (CPU, memory, time)
   - gVisor support for enhanced security

TECHNOLOGY STACK
================================================================================

FRONTEND:
- React 19
- Vite 6
- Tailwind CSS 4
- Zustand (state management)
- TanStack Query (data fetching)
- Framer Motion (animations)
- Monaco Editor (code editing)
- Boneyard-js (zero-layout-shift loading)

DATABASES:
- PostgreSQL 15 (relational data)

- Docker Compose (orchestration)
- Nginx (reverse proxy)
- Gunicorn (Django WSGI)
- Sentry (error tracking)
- Cloudinary (media storage)

================================================================================
QUICK START
================================================================================

1. Clone repository
2. Copy .env.example files and configure
3. Start backend: cd services && docker compose up -d --build
4. Start frontend: cd frontend && npm install && npm run dev
5. Access at http://localhost:5173

For detailed setup instructions, see: SETUP.txt
