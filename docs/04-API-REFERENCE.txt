================================================================================
CLASHCODE - API REFERENCE
================================================================================

BASE INFORMATION
================================================================================
Base URL: https://api.your-domain.com
API Version: v1
Authentication: JWT Bearer Token or HttpOnly Cookies
Content-Type: application/json

All API responses follow this format:
{
  "data": { ... },
  "error": null
}

Error response:
{
  "error": {
    "code": "API_ERROR",
    "message": "Description",
    "request_id": "uuid"
  }
}

================================================================================
AUTHENTICATION API
================================================================================

1. GET OAUTH URL (GitHub)
--------------------------
GET /api/auth/github/?state={random_string}

Response:
{
  "auth_url": "https://github.com/login/oauth/authorize?..."
}

2. GITHUB CALLBACK
-------------------
POST /api/auth/github/callback/
Content-Type: application/json

Body:
{
  "code": "github_oauth_code"
}

Response: JWT tokens in cookies or body

3. GET OAUTH URL (Google)
--------------------------
GET /api/auth/google/?state={random_string}

Response:
{
  "auth_url": "https://accounts.google.com/o/oauth2/auth?..."
}

4. GOOGLE CALLBACK
-------------------
POST /api/auth/google/callback/
Content-Type: application/json

Body:
{
  "code": "google_oauth_code"
}

5. REQUEST OTP
---------------
POST /api/auth/otp/request/
Content-Type: application/json

Body:
{
  "email": "user@example.com"
}

Rate Limit: 5 requests per minute

6. VERIFY OTP
--------------
POST /api/auth/otp/verify/
Content-Type: application/json

Body:
{
  "email": "user@example.com",
  "otp": "123456"
}

Response:
{
  "access": "jwt_access_token",
  "refresh": "jwt_refresh_token"
}

7. REFRESH TOKEN
-----------------
POST /api/auth/refresh/

Note: Refresh token sent automatically via HttpOnly cookie

8. LOGOUT
----------
POST /api/auth/logout/

Invalidates current tokens

CHAT API
================================================================================

1. WEBSOCKET CONNECTION
------------------------
URL: wss://api.your-domain.com/ws/chat/

JavaScript Example:
const ws = new WebSocket('wss://domain/ws/chat/');

ws.onopen = () => {
  // Authenticate
  ws.send(JSON.stringify({
    type: 'auth',
    token: 'jwt_token'
  }));
};

ws.onmessage = (event) => {
  const message = JSON.parse(event.data);
  console.log('New message:', message);
};

// Send message
ws.send(JSON.stringify({
  type: 'message',
  room_id: 'general',
  content: 'Hello!'
}));
